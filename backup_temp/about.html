About This Project

This project demonstrates DevOps automation for static websites using free cloud tools.

Technologies

  • Python 3.x
  • b2sdk (official Backblaze B2 SDK)
  • python-dotenv for secure environment variables
  • Git & GitHub Pages for hosting
  • HTML, CSS (Flex/Grid), JavaScript

Architecture

Website → Backblaze B2 → Disaster → Restore → GitHub Pages

How It Works

  1. Backup to Backblaze B2

Use this as a foundation for your own disaster recovery playbooks, or as a demo for portfolio and proof-of-concept automation.
